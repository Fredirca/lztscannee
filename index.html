LZT Fortnite Review Scanner

Static GitHub Pages app. Enter your token each visit, scan on demand, keep reports in your browser.

Security note

This version has no backend. Your API token is typed into the page and used directly by your browser. It is not saved to localStorage, not written to exported files, and is cleared when you reload/close the tab. Anyone with access to your device/browser DevTools while scanning could still see requests.
